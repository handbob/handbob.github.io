# handbob.github.io

### Requirements
* [Node](https://nodejs.org)   
* [React](https://react.dev)   
* [Vite](https://vitejs.dev)
* [Vitest](https://vitest.dev)

#### Unit tests
* Windows, macOS/Linux
```
npm run test
```
#### Run development application
* Windows, macOS/Linux
```
npm run dev
```
#### Build application
* Windows, macOS/Linux
```
npm run build
```
#### Run production application
* Windows, macOS/Linux
```
npm run preview
```
#### Deploy application
* Windows, macOS/Linux
```
npm run deploy
```
