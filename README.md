# handbob.github.io

### Requirements
* [Node](https://nodejs.org)   
* [React](https://react.dev)   
* [Vite](https://vitejs.dev)
* [Vitest](https://vitest.dev)
* [Playwright](https://playwright.dev)

#### Unit testing
* Windows, macOS/Linux
```
npm run test:unit
```
#### E2E testing
* Windows, macOS/Linux
```
npm run test:e2e
```
#### Run development application
* Windows, macOS/Linux
```
npm run dev
```
#### Build application
* Windows, macOS/Linux
```
npm run build
```
#### Run production application
* Windows, macOS/Linux
```
npm run preview
```
#### Deploy application
* Windows, macOS/Linux
```
npm run deploy
```
